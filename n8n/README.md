# n8n

This folder is organized as a small automation workspace:

- `workflows/`: exported n8n workflow JSON files.
- root config files: `package.json`, `package-lock.json`, `tsconfig.json`, `tsconfig.node.json`.

Workflow exports currently available:

- `workflows/agente-whatsapp.json`
- `workflows/hotel-kanban.json`
- `workflows/label-sync.json`
- `workflows/selective-sync.json`
